[부분 확인] IKI는 Dr. Ngo Duc Vuong의 40년 경험을 기반으로 베트남 전통 의학 지식을 디지털화한 'proprietary dataset'을 보유하고 있다고 주장하나, AI 모델의 정확도, 학습 데이터 규모, 알고리즘 차별성에 대한 구체적 검증이 부재하다. [미확인] 체질 분류 AI의 임상 검증 결과나 예측 정확도 지표가 제시되지 않아 '복제 불가능한 기술 자산'임을 독립적으로 확인할 수 없다. 현재 Founder-tier 단계의 앱으로, 대규모 사용자 피드백 루프를 통한 모델 개선이 이루어지지 않은 상태다. 경쟁사(베트남 내 헬스케어 앱, 글로벌 퍼스널라이제이션 플랫폼) 대비 기술적 우위를 입증할 벤치마킹 데이터가 필요하다.
A-3구독 모델의 지속성과 Churn 관리 능력데이터 부족▾
[확인] 연간 구독료 249K VND(약 $10)는 베트남 시장에서 접근 가능한 가격대이며, 2024년 $120K에서 2025년 $300K 목표는 2.5배 성장을 보여준다. [미확인] 그러나 MRR(월간 반복 매출), churn rate(해지율), LTV(고객 생애 가치), 재구독률 등 SaaS 비즈니스 핵심 지표가 전혀 제시되지 않아 수익 지속 가능성을 판단할 수 없다. 교육 콘텐츠와 커뮤니티를 통한 유기적 유입(CAC $0)은 강점이나, 유료 전환율과 장기 리텐션 데이터 없이는 '순이익 50%' 주장을 검증할 수 없다. PoC에서 LOTTE Mart 트래픽을 활용한 3만 설치 목표를 제시했으나, 오프라인 유입 사용자의 앱 정착률(D1/D7/D30 retention) 예측이 필요하다.
A-5기술 스택과 확장 가능성리스크 존재▾
[부분 확인] 3개 언어(베트남어/영어/일본어) 지원과 모바일 앱 출시는 확인되나, 백엔드 인프라, 클라우드 아키텍처, API 설계, 데이터 보안 체계에 대한 정보가 전무하다. [미확인] 팀 구성에서 AI/ML 엔지니어, 데이터 과학자, DevOps 인력이 명시되지 않아 CEO의 BrSE 경험만으로 AI 엔진을 자체 개발·운영하는지 의문이다. 현재 Founder-tier 사용자 규모에서 LOTTE Mart PoC로 3만 설치, LOTTE Members 앱 통합 시나리오로 확장할 경우 서버 부하, 실시간 추천 처리 능력, 데이터 파이프라인이 감당 가능한지 검증되지 않았다. Key person dependency가 높아 기술 리더 이탈 시 제품 유지보수에 치명적 리스크가 존재한다.

IKI는 베트남 전통 의학의 AI 개인화라는 독특한 포지셔닝과 10만 커뮤니티 기반 유기적 성장($0 CAC)으로 명확한 사회적 가치를 창출하고 있다. 2024년 $120K 매출 확인, 리트리트 1,000명 검증은 Product-Market Fit의 초기 신호다. 그러나 가장 큰 미검증 가정은 '오프라인 리트리트에서 검증된 체질 분류 → 구매 전환 모델'이 LOTTE Mart 같은 대형 유통 환경에서 즉각적 AOV 증가로 이어질지 여부다. 현재 앱은 Founder-tier 단계이며 3~6개월 내 3만 설치 목표는 공격적이다. 팀은 외부 검증 불가로 실행 역량에 의문이 있고, 순이익 50% 주장과 시장 규모 출처는 미확인이다.
