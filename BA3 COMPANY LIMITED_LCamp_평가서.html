Category A — SaaS/소프트웨어
A-1매출 및 고객 검증 (Revenue & Customer Validation)구체적 지표 부재▾
2024년 매출 발생을 주장하나 구체적 금액, 고객 수, 계약 형태가 미공개 상태 [미확인]. SaaS 모델의 핵심 지표인 MRR/ARR, 고객당 평균 계약 금액(ACV), Churn rate, Customer Acquisition Cost 등이 전혀 제시되지 않음 [미확인]. 'multiple brands and ecosystem partners'라는 표현은 정량적 검증이 불가능하며, 실제 유료 고객과 파일럿 고객의 구분도 불명확. Founder Institute 선발은 초기 검증으로 인정되나 실제 시장 견인력(traction)을 입증하기에는 불충분 [부분 확인].
A-2가격 전략 및 Unit Economics수익성 모델 불투명▾
SaaS 구독 및 데이터 라이선스 모델을 표방하나 구체적 가격대(티어별 요금제), 타겟 고객 세그먼트별 지불 의향(willingness to pay), Payback Period 등 Unit Economics 관련 정보가 전무 [미확인]. 경쟁사(Metric, Cube, Tabcut) 대비 가격 경쟁력이나 가치 기반 가격 책정(value-based pricing) 전략이 제시되지 않음. 데이터 라이선스 모델의 경우 API 접근 비용, 데이터 사용량에 따른 과금 체계 등이 불명확하여 확장 가능한 수익 모델인지 검증 불가 [미확인].
A-3제품 성숙도 및 기술 차별화AI 우위 근거 미약▾
IR 덱에서 'AI Assistant BÉ BA'를 주요 차별화 요소로 제시하나 기술 스택, AI 모델 성능 지표, 데이터 정확도 등 구체적 기술 우위 근거가 부족 [미확인]. 틱톡샵·쇼피·라자다 API 연동 깊이, 데이터 크롤링 인프라 안정성, 실시간 처리 능력에 대한 검증 정보 없음. '30% 매출 성장, 50% 시간 절감, 25% 제품 가시성 향상' 주장은 출처와 검증 데이터가 부재하여 신뢰도 낮음 [미확인]. 팀 구성에서 Data Engineer 1명만 명시되어 있어 AI/ML 엔지니어링 전문성이 제품 개발 속도와 품질을 지속 가능하게 만들 수 있는지 의문 [부분 확인].
A-4시장 확장 가능성 (TAM/SAM/SOM)시장 규모 검증 부족▾
베트남 이커머스 시장 성장은 명백하나 구체적 TAM/SAM/SOM 수치와 근거 자료 없음 [미확인]. BA3가 타겟하는 '브랜드 마케터'의 규모, 지불 능력, 현재 사용 중인 대안 솔루션(incumbent tools) 대비 전환 비용(switching cost) 분석이 부재. 베트남 외 동남아 시장(태국, 인도네시아 등) 확장 계획은 언급되나 각 시장별 플랫폼 점유율, 언어·규제 장벽, 로컬 경쟁 환경에 대한 구체적 전략 없음 [미확인]. 단순 '가능성' 수준에서 벗어나지 못함.
Category C — 데이터/AI 플랫폼
C-1데이터 파이프라인 및 인프라 안정성기술 역량 검증 필요▾
멀티 플랫폼(틱톡샵, 쇼피, 라자다) 데이터 수집·정제·분석 파이프라인의 안정성과 확장성에 대한 구체적 정보 부재 [미확인]. 각 플랫폼의 API 정책 변경, Rate Limit, 데이터 접근 제한에 대한 대응 전략이 불명확. 팀 구성에서 Data Engineer 1명만 명시되어 있어 대규모 데이터 처리와 실시간 분석을 안정적으로 수행할 수 있는 인프라 역량이 의심됨 [부분 확인]. 데이터 품질 관리(QA), 이상 탐지(anomaly detection), 백업 및 복구 전략 등 운영 안정성 관련 정보 전무 [미확인].
C-2AI 모델 성능 및 실용성감성 분석 정확도 미검증▾
'Consumer sentiment analysis'를 핵심 기능으로 제시하나 NLP 모델의 베트남어 처리 정확도, F1 Score, 실제 브랜드 의사결정에 미친 영향 등 성능 지표가 전혀 공개되지 않음 [미확인]. 소셜 플랫폼과 이커머스 리뷰 데이터의 텍스트 특성(슬랭, 이모티콘, 다국어 혼용)을 어떻게 처리하는지 불명확. 'AI Assistant BÉ BA'의 실제 사용자 만족도, 응답 정확도, 브랜드별 맞춤화 수준 등이 검증되지 않아 단순 챗봇 수준인지 실질적 인사이트 제공 도구인지 구분 불가 [미확인].
C-3데이터 보안 및 규제 대응정보 보안 정책 불명확▾
브랜드 민감 데이터(가격, 프로모션 전략, 매출 정보)를 다루는 플랫폼임에도 데이터 암호화, 접근 제어, GDPR/PDPA 준수 여부 등 보안 정책이 전혀 언급되지 않음 [미확인]. 베트남 개인정보보호법(Law on Personal Data Protection) 및 동남아 각국 데이터 규제 대응 방안 부재. 경쟁사 데이터 수집 과정에서의 법적 리스크(웹 크롤링 적법성, API 이용 약관 준수) 관리 전략이 불명확하여 플랫폼 운영 지속 가능성에 잠재적 위험 존재 [미확인].
